#### Git Interview Q&A
1 . What is Git and How do we use it in DevOps ?
#### Git is a distributed version control system used to track chanegs in source code during software development in DevOps. Git is used for version control, collaboratation, and to automate code deployments through CI/CD pipelines.

2 . How do you revert a commit that you made in your repository ?
#### Check Commit History : git log 
#### git revert (commit-ID)
#### To create a new commit that undoes the changs made by aprevious commit .
#### git push origin main

3 . What is branch in Repository ?
#### A branch in Git is a separate line of development. It allows developers to work on new features or fixes without affecting the main code.

4 . Explain me workflow of how you push your code from a local machine .
#### You push code from local to remote by committing your changes locally, then using git push to send those commits to a remote repository like GitHub or GitLab.
