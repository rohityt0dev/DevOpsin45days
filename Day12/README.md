### AWS Services that Every DevOps Engineers

#### What is aws ? 
AWS (Amazon Web Services) is a large cloud-computing platform from Amazon that provides on-demand computing power, storage, databases, networking, machine-learning tools, and many managed services over the internet.

##### Automation and Efficiency :
A DevOps engineer focuses on automation and improving efficiency in the software development and deployment process.

### AWS Services of DevOps 
##### Compute Services
1 . EC2 : Elastic Compute Cloud – Virtual servers for running apps

2 . Lambda : Serverless compute – Runs code on-demand without managing servers.
##### Storage & Networking
3 . EBS :	Elastic Block Store – Persistent block storage for EC2 instances, like virtual hard drives.

4 . S3 : Simple Storage Service – Scalable object storage for files, backups, websites.

5 . VPC :	Virtual Private Cloud – Your private virtual network (subnets, routing, security).
##### Security & Management
6 . IAM : Identity and Access Management – Controls user permissions and roles.

7 . KMS : Key Management Service – Manages encryption keys for data security.

8 . CloudTrail : Logs API calls and account activity for auditing.
##### Monitoring & Orchestration
9 . CloudWatch : Monitors metrics, logs, alarms for resources and apps.

10 . EKS : Elastic Kubernetes Service – Managed Kubernetes for container orchestration.
##### CI/CD Pipeline
11 . CodePipeline : Orchestrates build, test, deploy workflows.

​     CodeBuild : Compiles code, runs tests in the cloud. 

​     CodeDeploy : Automates app deployments to EC2, Lambda, on-premises.
##### Other Essentials
12 . AWS Config : Tracks resource configurations and compliance changes.

13 . Billing & Costing : Console/tools for usage tracking, budgets, cost allocation.
