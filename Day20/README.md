### GitHub Action Interview Questions : 
1 . What is the Purpose of Self-Hosted Runner in GitHub Actions ?

A self-hosted runner in GitHub Actions is a machine that we configure ourselves to execute CI/CD workflows instead of using GitHub’s managed runners.
It provides more control over the environment, allows access to private networks, helps reduce CI/CD cost and improves performance for custom workloads.

GitHub Actions vs Jenkins
GitHub Actions is a cloud-based CI/CD tool tightly integrated with GitHub, which makes it easy to automate build and deployment workflows using simple YAML configuration.

Jenkins is an open-source CI/CD server that provides more flexibility and plugin support but requires manual setup, maintenance and infrastructure management.

GitHub Actions is generally preferred for modern lightweight pipelines, while Jenkins is often used in enterprise environments for complex automation needs.
