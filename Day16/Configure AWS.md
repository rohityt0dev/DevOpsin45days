## Create AWS IAM User and Configure AWS in Local Machine .
In AWS Console :

Go → IAM

Users → Create User

Attach policy → AdministratorAccess (for learning)

Create user :

You will get:
> Access Key ID

> Secret Access Key

## Configure AWS in Local Machine
> aws configure

Enter :
> AWS Access Key ID: XXXXX
> 
> AWS Secret Access Key: XXXXX
> 
> Region: ap-south-1
> 
> Output format: json

**Now your local machine is connected to AWS*
