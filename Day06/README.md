##### Types of Permissions
| Permission | Symbol | Number | Meaning           |
| ---------- | ------ | ------ | ----------------- |
| Read       | r      | 4      | View file content |
| Write      | w      | 2      | Edit file         | 
| Execute    | x      | 1      | Run file          |

##### Users in Permission System 
 | User | Meaning      |
| ---- | ------------ |
| u    | Owner (user) |
| g    | Group        |
| o    | Others       |
| a    | All users    |
