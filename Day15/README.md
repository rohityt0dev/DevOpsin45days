## IaC :
IaC stands for Infrastructure as Code. It's a key DevOps practice for managing IT infrastructure through machine-readable code files, rather than manual setups.

#### Role in DevOps
IaC automates provisioning servers, networks, and cloud resources, enabling version control, testing, and repeatable deployments like application code.
This reduces errors, speeds up CI/CD pipelines, and ensures consistency across environments from dev to production.

# Terraform
Terraform is an infrastructure as code tool that lets you build, change, and version cloud and on-prem resources safely and efficiently.
